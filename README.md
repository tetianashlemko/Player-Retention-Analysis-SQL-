# Player-Retention-Analysis-SQL-
Завдання з курсу SQL 50 на LeetCode:  
Розрахувати частку гравців, які повернулися в гру наступного дня після їхнього першого входу.

Основні інструменти:  

*CTE (Common Table Expressions) для структурування запиту.  

*Агрегатні функції (MIN, COUNT) для визначення першої дати та загальної кількості.  

*Self-Join logic для порівняння активності у різні дні.  

*Scalar Subqueries для розрахунку знаменника (загальної кількості унікальних гравців) незалежно від фільтрів.  
  
  
Розрахунок утримання першого дня (Day 1 Retention). Показує відсоток новачків, які продовжили грати на другий день.  
Це головний показник того, чи подобається гра користувачам.
